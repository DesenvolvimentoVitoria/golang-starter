# golang-starter
Golang base project using fiber framework
 
### Folder Structure

here how golang-started manage directory:
```
|-- internal
|   |-- app
|   |-- config
|   |-- db
|   |   |-- localdb
|   |   |-- onlinedb
|   |-- middleware
|   |-- routes
|   |-- utils
|       |-- auth
|       |-- encryption
|       |-- response
|-- src
```

all global setting will place in internal folder
all module will place in src
